# Library

> Biblioteca compartida de artículos, papers, blogs, videos y cualquier recurso valioso.

## Cómo agregar un recurso

1. Copia `_template.md` a un archivo nuevo
2. Nombre del archivo: `YYYY-MM-titulo-corto.md`
   - Ejemplo: `2026-05-attention-is-all-you-need.md`
   - Ejemplo: `2026-05-blog-karpathy-llm-zero-to-hero.md`
3. Llena el bloque inicial (link, autor original, por qué lo subiste, tags)
4. Si lo leíste, agrega tu sección de notas firmada con `**[D]**` o `**[M]**`

## Cómo comentar lo que el otro subió

Al final del archivo, agrega tu propia sección:

```markdown
## Notas de [tu inicial]

**[M]** Lo más interesante para mí fue...
```

No edites las notas del otro, solo agregas las tuyas debajo. Es un hilo asíncrono.

## Tags sugeridos

Para que sea fácil filtrar después:

- `paper` — paper académico
- `blog` — artículo de blog
- `video` — video o curso
- `book` — libro o capítulo
- `tool` — herramienta o framework
- `transformers` `llm` `cv` `mlops` `theory` — temas
- `must-read` — recurso prioritario según el mentor

## Convenciones

- El idioma del archivo es libre (español o inglés según el contenido original)
- Si es un paper técnico, incluye al menos el abstract en tus palabras
- No es un resumen completo — son tus notas y reflexiones, no un sustituto del original
